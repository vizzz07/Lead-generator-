\# Corporate Travel Lead Generator 🚀



This is a Python + Flask-based lead generation tool that helps you collect company contact details such as:



\- Company Name

\- Website

\- Emails

\- Phone Numbers

\- Contact Pages



The tool runs locally on your system.



\---



\## 🔧 Features



\- Search companies by industry and city

\- Extract public contact information from websites

\- Export results to Excel

\- No API required (basic version)



\---



\## ⚙️ Requirements



\- Python 3.x

\- pip

pip install -r requirements.txt
